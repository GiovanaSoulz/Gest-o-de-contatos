# Gestao de contatos
Uma gestão de contatos simples mobile no expoGo 

#No cmd ->
mkdir <nome da pasta>
cd <nome da pasta>
npm install -g expo-cli
npx create-expo-app <nome da pasta> --template blank-typescript
cd <nome da pasta>
npm run android
